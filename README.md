# trxconcept.cl

Sitio web estático de marketing para TRX Concept, negocio de entrenamiento personal en Santiago, Chile. Publicado en GitHub Pages.

## Estructura

```
index.html                      → /
sobre-mi/index.html             → /sobre-mi
servicios/index.html            → /servicios
preguntas-frecuentes/index.html → /preguntas-frecuentes
assets/css/style.css            → hoja de estilos compartida
assets/img/                     → imágenes WebP
```

## Desarrollo local

Sin herramientas de build. Para previsualizar, abre cualquier archivo HTML o córrelo en un servidor local.

Edita los archivos y haz push para desplegar directamente.

## Notas

- Todos los cambios en el header/nav deben aplicarse en **todas las páginas**.
